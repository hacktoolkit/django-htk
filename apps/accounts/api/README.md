# Api

## Functions
- **`suggest`** (api/views.py) - This API endpoint supports User autocomplete
- **`update`** (api/views.py) - Updates a User or UserProfile
- **`username`** (api/views.py) - Update a User's username
- **`password`** (api/views.py) - Update a User's password
- **`avatar`** (api/views.py) - Update a User's avatar to the specified type
- **`follow`** (api/views.py) - Follow another user
- **`unfollow`** (api/views.py) - Unfollow another user
