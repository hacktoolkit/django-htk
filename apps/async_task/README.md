# Async_Task

- **`build_async_task_result`** (async_task/utils.py) - Builds an Async Task result from JSON
- **`extract_async_task_result_json_values`** (async_task/utils.py) - Companion function to perform the inverse of `build_async_task_result()`
- **`async_download_result`** (async_task/views.py) - View to download the result of an async task as a file
