# Oembed

## Functions
- **`get_oembed_html`** (oembed/utils.py) - Gets the oEmbed HTML for a URL, if it is an oEmbed type
- **`get_oembed_html_for_service`** (oembed/utils.py) - Returns the oEmbed HTML for `service` (YouTube, Vimeo, etc)
- **`get_oembed_type`** (oembed/utils.py) - Determines the type of oEmbed this URL is, if it exists
